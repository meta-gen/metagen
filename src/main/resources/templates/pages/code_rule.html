3. 코드 규칙 관리

• 설명: 다양한 언어의 코드 네이밍 및 주석 규칙을 설정하고 관리할 수 있는 화면.

• 기능:

  • 규칙에 따른 네이밍 생성
  • 주석 자동 생성 기능
  • Excel 템플릿으로 규칙 업로드 및 다운로드
  • 버전 관리 및 규칙 히스토리 확인
  • 관리자 승인 프로세스 (활성화 여부 선택 가능)
  • Prefix, Subfix 설정 및 변수 타입별 규칙 지정
  • 조회, 다량조회, 수정, 삭제 기능별로 Prefix, Subfix 세분화
  • 자동 생성된 규칙을 설계서 및 테스트 시나리오에 적용 가능

• 추가 요구사항:

  • 관리자가 규칙을 완성하기 전까지 화면 비활성화
  • 다양한 언어와 변수 타입 대응 (Java, Python, C++ 등)
  • 생성된 규칙을 설계서 및 테스트 시나리오 관리 페이지와 연동
  • 주석 템플릿 커스터마이징 지원

• 논의사항:

  • 배치 작업으로 규칙을 대량 처리할 필요성
  • 향후 다양한 언어와 코드 스타일 확장 여부
  • 주석 템플릿의 동적 규칙 설정 범위
