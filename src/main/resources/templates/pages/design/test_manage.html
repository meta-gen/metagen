테스트 시나리오 관리

• 설명: 테스트 시나리오를 생성, 관리하며 다운로드할 수 있는 화면.

• 기능:

  • Excel 템플릿을 활용한 테스트 시나리오 데이터 업로드 및 다운로드
  • 자동 생성된 테스트 케이스 확인
  • 생성된 테스트 시나리오에 주석 및 메소드명 자동 반영
  • 버전 관리 (각 테스트 시나리오의 수정 이력 관리)

• 추가 요구사항:

  • 공유된 파일들만 다운로드 기록에 저장
  • PDF, Excel 등의 형식으로 다운로드
  • 작업 전 규칙 기반의 메소드 및 주석 설정 필요 (코드 규칙 관리와 연계)
  • 내부망 환경에 최적화된 다운로드 및 업로드 작업

• 논의사항:

  • 배치 작업으로 템플릿 데이터를 동기화할 필요성
  • 다운로드된 테스트 시나리오의 수정 여부 관리
  • 자동화 수준 (데이터 입력부터 테스트 시나리오 완성까지)
