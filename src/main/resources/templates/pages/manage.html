5. 사용자 관리

• 설명: MetaGen 사용자 계정을 생성하고 관리할 수 있는 관리자 전용 화면. 사용자 권한 및 역할을 설정하고 시스템 접근을 제어할 수 있습니다.

• 기능:

  • 사용자 계정 관리
    • 사용자 계정 생성, 수정, 삭제
    • 비밀번호 초기화 및 계정 활성화/비활성화 설정
  • 권한 및 역할 관리
    • 역할(Role) 및 권한(Permission) 설정
    • 관리자(Admin), 일반 사용자(User), 게스트(Guest) 등 역할 정의
    • 특정 메뉴 접근 제한 설정
  • 액세스 로그 관리
    • 사용자 활동 로그 확인 (로그인/로그아웃, 데이터 수정 등)
    • 시간별, IP별, 사용자별 로그 조회 기능
  • 그룹 관리 (Optional)
    • 사용자 그룹 생성 및 관리
    • 그룹별 권한 설정
  • 승인 프로세스
    • 새 사용자 등록 시 관리자 승인 여부 결정
    • 승인 대기 사용자 목록 관리
    • 승인 시스템 사용 여부 설정
  • 코드 규칙 관리
    • 새로운 코드 규칙 등록
    • 기존 규칙 수정 및 삭제
    • 설계서 및 시나리오에 연동될 기본 규칙 설정
  • 템플릿 언어 설정
    • 기본 템플릿 언어 지정 (Java, Python, C++, etc.)
    • 다양한 언어 확장을 위한 템플릿 관리
  • 알림 설정
    • 계정 관련 알림 제공 (비밀번호 만료, 승인 대기 등)

• 추가 요구사항:

  • 관리자 전용 메뉴로 접근 제한
  • 비밀번호는 반드시 암호화하여 저장 (e.g., BCrypt)
  • 다국어 지원 가능하도록 UI 설계
  • 승인 시스템 및 코드 규칙 연동 설정은 관리자만 접근 가능

• 논의사항:

  • 비밀번호 정책 (길이, 특수문자 포함 여부 등)
  • 사용자 그룹과 역할 관리의 세분화 필요성
  • 템플릿 언어 설정 시 확장 가능한 언어의 범위
  • 기본 코드 규칙과 사용자 정의 규칙 간의 충돌 처리 방안
